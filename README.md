# CarbonLens
## Personal Lifestyle Emission Analyzer

CarbonLens is a full-stack sustainability intelligence platform designed to help individuals measure, analyze and reduce  their personal carbon footprint based on the everyday lifestyle activities.

## Major Emission Sources
-Transportation
-Electricity Usage
-Consumption / Spending

## Features
- CO₂ estimation per activity
- Weekly, monthly, annual analytics
- Emission hotspot detection
- Sustainability scoring
- Green Credit reward system

Community impact simulation
---

# How It Works

Users input lifestyle activities.
The backend calculates emissions using verified emission factors.
Analytics identify trends and hotspots.
The system rewards emission reduction through a Green Credit mechanism.
