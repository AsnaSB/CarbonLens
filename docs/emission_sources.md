# Emission Factor Sources

Transport emission factors:
- Based on average DEFRA-style public datasets
- Car ~0.21 kg CO₂ per km
- Bus ~0.05 kg CO₂ per km
- Train ~0.04 kg CO₂ per km
- Flight ~0.15 kg CO₂ per km

Electricity:
- Default grid factor: 0.82 kg CO₂ per kWh (India average estimate)

Spending:
- Approximated emission intensity per currency unit
- Used for relative comparison, not scientific precision

Note:
Values are configurable via:
app/data/emission_factors.json
